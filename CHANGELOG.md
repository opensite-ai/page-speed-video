# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-12

### Added
- Initial release of @page-speed/video
- HLS streaming support with hls.js
- Native HLS support for Safari/iOS
- Progressive MP4 fallback for unsupported browsers
- Transform API integration for automatic HLS conversion
- OptixFlow poster optimization support
- Two processing strategies: optimistic and poll
- TypeScript support with full type definitions
- Tree-shakeable ESM exports
- CJS compatibility
- UMD build for CDN usage
- Debug logging capabilities
- VideoDefaults component for global configuration
